yuzuemon/dotfiles
====

## information

This repository is my dotfiles.

* Bazaar
* Git
* Vim
* tmux
* Zsh

except ~/bin

## How to Setup

1. git clone git@github.com:yuzuemon/dotfiles.git
1. cd ./dotfiles
1. ./setup.sh
1. cd ./.vim
1. git clone https://github.com/Shougo/neobundle.vim
1. cd -
1. vi ./.vimrc.neobundle
1. :NeoBundleInstall!
