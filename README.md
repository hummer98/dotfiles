yuzuemon/dotfiles
====

## Information

This repository is my dotfiles.

* Git
* Vim
  * with neobundle.vim
* tmux
* zsh
* and more...

see more [setup.sh](/setup.sh) 's DOT_FILES

## How to Setup

```sh
git clone https://github.com/yuzuemon/dotfiles.git
cd ./dotfiles
./setup.sh
```

## How to NeoBundle's Install

```sh
cd ~/.vim/
git clone https://github.com/Shougo/neobundle.vim
vi ~/.vimrc.neobundle
:NeoBundleInstall!
```
